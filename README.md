# quantum-walk-5cycle-qkd-protocol
Reference Python implementation of the quantum walk-based quantum direct communication protocol introduced in Phys. Rev. A 107, 022611 (2023). Includes simulations of ideal recurrence dynamics and the effects of amplitude damping and depolarizing noise.

**Reference Python implementation of the quantum walk-based quantum direct communication (QDC) protocol introduced in:**

**S. S. Panda, P. A. Ameen Yasir, and C. M. Chandrashekar**,
*"Quantum direct communication protocol using recurrence in k-cycle quantum walks,"*
**Physical Review A 107, 022611 (2023).**
DOI: https://doi.org/10.1103/PhysRevA.107.022611

---

## Overview

This repository provides the numerical simulations used to study a quantum direct communication protocol based on the recurrence properties of discrete-time quantum walks (DTQWs) on finite cycle graphs. The implementation reproduces the principal results presented in the associated publication and investigates the influence of realistic quantum noise channels on the protocol.

The code includes three self-contained simulation modules:

* **Ideal 5-cycle DTQW (Noiseless):** Evolution and recurrence dynamics of the protocol without environmental decoherence.
* **Amplitude Damping Noise:** Analysis of the protocol under amplitude damping acting on the coin degree of freedom.
* **Depolarizing Noise:** Simulation of the protocol in the presence of depolarizing noise and evaluation of its effect on recurrence and mutual information.

The repository is intended as a **reference implementation** accompanying the published work and as a starting point for further studies of quantum walk-based communication protocols.

---

## Features

* Discrete-time quantum walk simulation on finite cycle graphs.
* Implementation of the coin operator for k-cycle recurrence.
* Quantum channel models:

  * Amplitude damping
  * Depolarizing noise
* Calculation of:

  * Return probability (P(t))
  * Coin-position mutual information (I_\mu(t))
* Automatic generation of publication-quality figures.

## Requirements

The simulations require Python 3 and the following packages:

* `numpy`
* `matplotlib`
* `pandas`
* `qutip`

Install the dependencies with:

```bash
pip install numpy matplotlib pandas qutip
```

Optionally, a LaTeX installation can be used for high-quality plot labels. If LaTeX is unavailable, the code automatically falls back to standard text rendering.

---

## Usage

Run the main simulation script:

```bash
python quantum_walk_simulations.py
```

The script contains three independent simulation blocks, each of which can be executed separately to reproduce the corresponding numerical results.
